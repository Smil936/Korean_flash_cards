Install on Phone
iPhone: Open in Safari → Share → Add to Home Screen
Android: Open in Chrome → ⋮ → Add to Home Screen
Sheets Setup Guide
  1. Create a Google Sheet, add a tab called Log
  2. Row 1 headers: Word | Result | Timestamp
  3. Share → Anyone with the link can edit
  4. Google Cloud Console → enable Sheets API → create API Key
  5. Restrict key to Google Sheets API only
  6. Paste Sheet ID (from the URL) and key above
